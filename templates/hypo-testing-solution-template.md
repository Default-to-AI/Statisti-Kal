# תבנית מקיפה לפתרון בדיקת השערות

מסמך זה מציג את השלבים הפורמליים בבדיקת השערות, משולבים בדוגמה מספרית. נתוני הדוגמה הפיקטיבית: מפעל מייצר סוללות עם תוחלת אורך חיים היסטורית של $\mu_0 = 1000$ שעות וסטיית תקן ידועה של $\sigma = 50$ שעות. הוכנס תהליך ייצור חדש, ונלקח מדגם של $n = 100$ סוללות. ממוצע המדגם שהתקבל הוא $\bar{X} = 1012$ שעות. רמת המובהקות הנדרשת: $\alpha = 0.05$.

## 1. ניסוח השערות המחקר

**הסבר פורמלי:** הגדרת השערת האפס ($H_0$), המניחה היעדר שינוי, מול השערת המחקר ($H_1$), המייצגת את טענת החוקר.

**תבנית כללית:**
*   **מבחן דו-צדדי:** $H_0: \mu = \mu_0 \quad \text{vs} \quad H_1: \mu \neq \mu_0$
*   **מבחן ימני:** $H_0: \mu \le \mu_0 \quad \text{vs} \quad H_1: \mu > \mu_0$
*   **מבחן שמאלי:** $H_0: \mu \ge \mu_0 \quad \text{vs} \quad H_1: \mu < \mu_0$

**יישום בדוגמה:** אנו בודקים האם תהליך הייצור החדש שינה את תוחלת אורך החיים (ללא כיוון מוגדר מראש), לכן נבצע מבחן דו-צדדי:

$$H_0: \mu = 1000 \quad \text{vs} \quad H_1: \mu \neq 1000$$

## 2. התפלגות הסטטיסטי ושגיאת התקן (SE)

**הסבר פורמלי:** קביעת ההתפלגות של ממוצע המדגם $\bar{X}$ תחת הנחת $H_0$, וחישוב שגיאת התקן המייצגת את הפיזור של ממוצעי המדגמים.

**תבנית כללית:** כאשר השונות $\sigma^2$ ידועה (או $n > 30$): $\bar{X} \sim N\left(\mu_0, \frac{\sigma^2}{n}\right)$

$$SE = \frac{\sigma}{\sqrt{n}}$$

(אם השונות אינה ידועה, נשתמש בסטיית התקן המדגמית $S$ ובהתפלגות $t$ עם $n-1$ דרגות חופש).

**יישום בדוגמה:** השונות ידועה ($\sigma = 50$), גודל המדגם $n = 100$.

$$SE = \frac{50}{\sqrt{100}} = \frac{50}{10} = 5$$

לכן, תחת $H_0$: $\bar{X} \sim N(1000, 5^2)$

## 3. קביעת כלל החלטה (אזורי דחייה וקבלה)

**הסבר פורמלי:** הגדרת הערכים הקריטיים והקבוצות הפורמליות של אזור הדחייה ($C$) ואזור הקבלה ($C^c$). במידה והסטטיסטי שייך ל-$C$, נדחה את $H_0$ בהסתברות $\alpha$.

**תבנית כללית (ניסוח כקבוצות):**

*   **מבחן דו-צדדי:**
    $$C = \left\{ \bar{X} \mid \bar{X} \le \mu_0 - Z_{1-\alpha/2}\cdot SE \quad \lor \quad \bar{X} \ge \mu_0 + Z_{1-\alpha/2}\cdot SE \right\}$$
    $$C^c = \left\{ \bar{X} \mid \mu_0 - Z_{1-\alpha/2}\cdot SE < \bar{X} < \mu_0 + Z_{1-\alpha/2}\cdot SE \right\}$$

*   **מבחן ימני:**
    $$C = \left\{ \bar{X} \mid \bar{X} \ge \mu_0 + Z_{1-\alpha}\cdot SE \right\}$$
    $$C^c = \left\{ \bar{X} \mid \bar{X} < \mu_0 + Z_{1-\alpha}\cdot SE \right\}$$

*   **מבחן שמאלי:**
    $$C = \left\{ \bar{X} \mid \bar{X} \le \mu_0 - Z_{1-\alpha}\cdot SE \right\}$$
    $$C^c = \left\{ \bar{X} \mid \bar{X} > \mu_0 - Z_{1-\alpha}\cdot SE \right\}$$

**יישום בדוגמה (מבחן דו-צדדי, $\alpha = 0.05$):** נחשב את הערכים הקריטיים:

$$C_1 = 1000 - 1.96 \cdot 5 = 990.2$$

$$C_2 = 1000 + 1.96 \cdot 5 = 1009.8$$

**כלל החלטה מוגדר קבוצות:**

$$C = \{ \bar{X} \mid \bar{X} \le 990.2 \quad \lor \quad \bar{X} \ge 1009.8 \}$$

$$C^c = \{ \bar{X} \mid 990.2 < \bar{X} < 1009.8 \}$$

## 4. חישוב סטטיסטי, P-Value והכרעה

**הסבר פורמלי:** התקננת ממוצע המדגם לציון תקן, חישוב ההסתברות לקבל תוצאה קיצונית מזו שנצפתה תחת $H_0$, והסקת מסקנות.

**תבנית כללית:**

$$Z_{stat} = \frac{\bar{X} - \mu_0}{SE}$$

**P-Value דו-צדדי:** $2 \cdot (1 - \Phi(|Z_{stat}|))$

**יישום בדוגמה:** ממוצע המדגם בפועל הוא $\bar{X} = 1012$.

$$Z_{stat} = \frac{1012 - 1000}{5} = \frac{12}{5} = 2.4$$

$$P-Value = 2 \cdot P(Z > 2.4) = 2 \cdot (1 - 0.9918) = 2 \cdot 0.0082 = 0.0164$$

**הכרעה:** מכיוון ש-$\bar{X} = 1012$ שייך לאזור הדחייה ($\bar{X} \in C$), וכן $P-Value (0.0164) < \alpha (0.05)$, אנו דוחים את השערת האפס.

**מסקנה מילולית:** ברמת מובהקות של 5%, קיימת עדות סטטיסטית מספקת לכך שתוחלת אורך החיים של הסוללות השתנתה בעקבות תהליך הייצור החדש.

## 5. טעות מסוג שני ($\beta$) ועוצמת המבחן ($1-\beta$)

**הסבר פורמלי:**
*   $\beta$: ההסתברות שממוצע המדגם ייפול באזור הקבלה ($C^c$) כאשר במציאות $H_1$ נכונה (עבור תוחלת ספציפית $\mu_1$).
*   $1-\beta$ (**עוצמה**): ההסתברות לזהות נכונה שינוי שקיים במציאות ולדחות את $H_0$.

**תבנית כללית (למבחן דו-צדדי):**

$$\beta = P(\bar{X} \in C^c \mid \mu = \mu_1) = P(C_1 < \bar{X} < C_2 \mid \mu = \mu_1) = \Phi\left(\frac{C_2 - \mu_1}{SE}\right) - \Phi\left(\frac{C_1 - \mu_1}{SE}\right)$$

$$\text{Power} = 1 - \beta$$

**יישום בדוגמה:** נניח שבמציאות התהליך החדש האריך את תוחלת החיים ל-$\mu_1 = 1015$.
אנו מחשבים את ההסתברות ש-$\bar{X}$ ייפול באזור הקבלה ההיסטורי ($990.2 < \bar{X} < 1009.8$) תחת ההתפלגות החדשה $\bar{X} \sim N(1015, 5^2)$:

$$\beta = P(990.2 < \bar{X} < 1009.8 \mid \mu = 1015)$$

$$\beta = P\left(\frac{990.2 - 1015}{5} < Z < \frac{1009.8 - 1015}{5}\right)$$

$$\beta = P(-4.96 < Z < -1.04)$$

$$\beta = \Phi(-1.04) - \Phi(-4.96) \approx 0.1492 - 0 = 0.1492$$

**עוצמת המבחן:**

$$1 - \beta = 1 - 0.1492 = 0.8508$$

**הסבר:** יש סיכוי של כ-85% שהמבחן שלנו יצליח לזהות בוודאות את השיפור באורך החיים (אם הוא אכן עלה ל-1015) ולדחות את השערת האפס ההיסטורית.

## 6. רווח סמך (Confidence Interval)

**הסבר פורמלי:** בניית טווח ערכים שברמת ביטחון של $1-\alpha$ יכיל את פרמטר האוכלוסייה האמיתי $\mu$, על בסיס נתוני המדגם.

**תבנית כללית:**

$$CI_{1-\alpha} = \left[ \bar{X} - Z_{1-\alpha/2} \cdot SE, \quad \bar{X} + Z_{1-\alpha/2} \cdot SE \right]$$

**יישום בדוגמה (רמת ביטחון 95%):**

$$CI = \left[ 1012 - 1.96 \cdot 5, \quad 1012 + 1.96 \cdot 5 \right]$$

$$CI = \left[ 1012 - 9.8, \quad 1012 + 9.8 \right]$$

$$CI = [1002.2, \quad 1021.8]$$

**מסקנה מילולית:** אנו בטוחים ברמת ביטחון של 95% שתוחלת אורך החיים של הסוללות המיוצרות בתהליך החדש נמצאת בין 1002.2 ל-1021.8 שעות.
מכיוון שערך השערת האפס ($1000$) אינו מוכל בתוך רווח הסמך, זוהי הוכחה שקולה לכך שיש לדחות את $H_0$ ברמת מובהקות של $\alpha=0.05$.
