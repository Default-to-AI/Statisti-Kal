להלן תבניות פורמליות לניסוח המסקנה הסטטיסטית הסופית, מחולקות לפי שלוש הגישות ולפי החלטת הדחייה או אי-הדחייה של השערת האפס ($H_0$).

התבניות משלבות מקומות ייעודיים להזנת הנתונים הספציפיים של התרגיל (מסומנים בסוגריים מרובעים), ומותאמות לשימוש עם התפלגות $Z$ או $t$.

---

### 1. גישת סטטיסטי המבחן (Test Statistic)

גישה זו מתבססת על השוואת סטטיסטי המבחן המחושב מתוך המדגם לערך הקריטי מהטבלה הסטטיסטית.

**במקרה של דחיית $H_0$:**

> "ברמת מובהקות של $\alpha$ = **[הזן ערך]**, סטטיסטי המבחן שחושב, **[$Z$ או $t$]** = **[הזן את ערך הסטטיסטי]**, נופל באזור הדחייה מכיוון שהוא **[גדול / קטן / קיצוני]** מהערך הקריטי **[$Z_{crit}$ או $t_{crit}$]** = **[הזן את הערך הקריטי מהטבלה]**. לכן, אנו דוחים את השערת האפס ($H_0$) ומסיקים סטטיסטית כי **[הזן את מסקנת המחקר במילים, למשל: ממוצע השכר אכן עלה]**."

**במקרה של אי-דחיית $H_0$:**

> "ברמת מובהקות של $\alpha$ = **[הזן ערך]**, סטטיסטי המבחן שחושב, **[$Z$ או $t$]** = **[הזן את ערך הסטטיסטי]**, אינו נופל באזור הדחייה מכיוון שהוא אינו **[גדול / קטן / קיצוני]** מהערך הקריטי **[$Z_{crit}$ או $t_{crit}$]** = **[הזן את הערך הקריטי מהטבלה]**. לכן, אין בידינו מספיק עדות סטטיסטית כדי לדחות את השערת האפס ($H_0$), ולא ניתן להסיק כי **[הזן את טענת החוקר במילים]**."

---

### 2. גישת אזור הדחייה / הערך המקורי ($\bar{X}$ או $\hat{p}$)

גישה זו מתבססת על השוואת ממוצע (או פרופורציית) המדגם לערך הסף הקריטי במונחי יחידות המידה המקוריות של הבעיה.

**במקרה של דחיית $H_0$:**

> "ברמת מובהקות של $\alpha$ = **[הזן ערך]**, הערך שהתקבל במדגם, $\bar{X}$ = **[הזן את ממוצע המדגם]**, חצה את ערך הסף הקריטי $\bar{X}_{crit}$ = **[הזן את ערך הסף שחושב]** ושייך לאזור הדחייה. לפיכך, אנו דוחים את השערת האפס ($H_0$) ברמת ביטחון של **[הזן רמת סמך באחוזים, למשל 95%]** ומקבלים את השערת החוקר כי **[הזן את מסקנת המחקר במילים]**."

**במקרה של אי-דחיית $H_0$:**

> "ברמת מובהקות של $\alpha$ = **[הזן ערך]**, הערך שהתקבל במדגם, $\bar{X}$ = **[הזן את ממוצע המדגם]**, אינו שייך לאזור הדחייה, שכן הוא אינו חורג מערך הסף הקריטי $\bar{X}_{crit}$ = **[הזן את ערך הסף שחושב]**. לפיכך, אנו לא דוחים את השערת האפס ($H_0$); נתוני המדגם אינם מספקים תמיכה מובהקת לטענה כי **[הזן את טענת החוקר במילים]**."

---

### 3. גישת מובהקות התוצאה (P-Value)

גישה זו משווה בין ההסתברות לקבל תוצאה קיצונית כמו זו שבמדגם (תחת הנחת $H_0$) לבין רמת המובהקות שקבע החוקר מראש.

**במקרה של דחיית $H_0$:**

> "מכיוון שהתקבל ערך מובהקות (P-Value = **[הזן את ההסתברות שחושבה]**) אשר קטן מרמת המובהקות שנקבעה, $\alpha$ = **[הזן ערך]**, התוצאה מובהקת סטטיסטית. לפיכך, אנו דוחים את השערת האפס ($H_0$) ומסיקים כי **[הזן את מסקנת המחקר במילים]**."

**במקרה של אי-דחיית $H_0$:**

> "מכיוון שהתקבל ערך מובהקות (P-Value = **[הזן את ההסתברות שחושבה]**) אשר גדול מרמת המובהקות שנקבעה, $\alpha$ = **[הזן ערך]**, התוצאה אינה מובהקת סטטיסטית. משום כך, אנו לא דוחים את השערת האפס ($H_0$), והמדגם לא סיפק עדות חזקה מספיק כדי לאשש את הטענה כי **[הזן את טענת החוקר במילים]**."
